# Auto Clicker

一个简单的自动点击器程序，支持自定义点击间隔，并通过键盘快捷键控制。以打包为exe文件。
![流水线](流水线.jpg)

## 功能
- 按下退格键（Backspace）开始模拟鼠标左键单击。
- 在点击过程中，按下任意键停止点击。
- 可通过输入框动态更新点击间隔时间（单位：秒）。

## 使用方法
1. 运行程序。
2. 在输入框中输入点击间隔时间（默认为6秒）。
3. 点击“更新时间”按钮应用新的间隔时间。
4. 按下退格键开始自动点击。
5. 在点击过程中，按下任意键停止。
